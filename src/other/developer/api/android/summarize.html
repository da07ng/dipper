概述-Runtime SDK

简介

ArcGIS Runtime SDK for Android是Esri 为开发者提供的移动应用开发包。只要注册了 Esri 全球账号，就可以免费下载各个版本的开发包以及其它相关资料。ArcGIS Runtime SDK for Android的API包含了200多个类和接口，覆盖了地图、数据、功能的展示和操作等方方面面，还可调用ArcGIS for Server提供的强大制图、地理编码、地理处理和自定义功能和Portal for ArcGIS上的地图，API帮助中都写得非常详细，使用Java语言和Eclipse IDE构建Android设备上的应用程序，并将它们部署到Android手机和平板等设备上。

功能介绍

基础地图服务：支持ArcGIS在线数据源、ArcGIS离线数据源（自10.2版本起），以及第三方数据源的加载。

空间查询服务：提供本地数据和在线数据对应的API，实现空间查询。

空间统计服务：基于空间查询的结果，进行信息的统计分析。

定位服务：用于提供从地址到经纬度坐标或者从经纬度坐标到地址的转换服务

标注服务：可以对图片，点，线，面，文字等多类要素进行样式设置，以实现地图标注。

分级渲染服务-点：通过点的渲染方式进行分级结果展示。

分级渲染服务-面：通过面的渲染方式进行分级结果展示。

统计信息展示服务：结合渲染功能，通过图表、柱状图等多种形式就是信息展示。

时态展示服务：时态服务包括两种类型，一种是动态的时态服务，一种是要素的时态服务。

大数据渲染服务：通过分级渲染器（ClassBreaksRenderer）对大数据进行分级渲染。
