概述-IOS API

简介

新华社地图iOS API是一套由Objective-C开发的应用程序接口，支持开发丰富的移动地图应用，可以运行在iPad、iPhone 和 iPod touch 上，本指南提供了入门指导和接口介绍。

功能介绍

基础地图服务:图层（layer）在地图容器（MapView）中是按顺序加载的,默认第一个图层为基础底图，基础底图非常重要，它决定了地图应用需要的基本参数，包括投影类型、显示范围、比例尺等

空间查询服务：提供本地数据和在线数据对应的API，实现空间查询。查询一般分为属性查询和空间查询，属性查询通过构造标准的SQL语句实现，而空间查询比较特殊，需要构建几何范围并指定空间关系。

空间统计服务：基于空间查询的结果，进行信息的统计分析。

定位服务：定位能力是智能设备的标配，SDK提供了对定位信息的管理功能，用户可以通过LocationDisplayManager来进行定位操作

标注服务：包括所有叠加到地图的内容，如标注、矢量图形元素(包括折线和多边形和圆)、信息窗体等。覆盖物具有地理坐标，当地图拖动或缩放时，它们会对应的移动。

分级渲染服务-点：英文为Render，可以用一种或多种符号实现空间要素数据（点、线、面）的批量符号化，以下通过ClassBreakRenderer接口实现点要素数据的分级渲染，分级渲染器定义了一组符号和数值属性值之间的映射，其中ClassBreak是最基本的单元，需要指定该分级区间的最小、最大值和符合样式。

分级渲染服务-面：通过ClassBreakRenderer接口实现面要素的分级渲染，分级渲染器定义了一组符号和数值属性值之间的映射，其中ClassBreak是最基本的单元，需要指定该分级区间的最小、最大值和符合样式。

时态展示服务：地图API支持可视化时态数据，通过时间渲染器（temporal render）,可以很容易地展示包含时间属性的地图图层（Time-aware Layers）随时间发生的变化，并且可以按时间过滤数据。

大数据渲染服务：通过分级渲染器（ClassBreaksRenderer）对大数据进行分级渲染。
