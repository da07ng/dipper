标注服务

简介

Symbol定义了如何在GraphicLayer上显示点、线、面和文本，符号定义了几何对象的所有非地理特征方面的外观，包括图形的颜色，边框线宽度，透明度等等。

ArcGIS定义了如下几种标注方式：

  • TextSymbol：地图上加载自定义文字。
  • PictureMarkerSymbol：标注表示地图上的点，可自定义标注的图片。
  • LineSymbol：表示地图上的折线。
  • PictureFillSymbol：填充符号用于绘制图层上的多边形图形。PictureFillSymbol填充图像中的重复图案。另外，符号可以有一个可选的轮廓，这是由一个线符号定义的。限制：PictureFillSymbol不会在Firefox2.x正常工作。它支持所有浏览器。
  • SimpleFillSymbol：填充符号用于绘制图层上的多边形图形。SimpleFillSymbol可以是固体的，透明的，或几种交叉影线图案中的一个。另外，符号可以有一个可选的轮廓，这是由一个线符号定义的。限制：当使用火狐2.x，只有STYLE_SOLID和STYLE_NULL是有效的。它支持所有现代浏览器。
  • SimpleMarkerSymbol：标记符号用于绘制图形层上的单点和多点。SimpleMarkerSymbol将点显示为一个简单的形状，例如圆形。另外，符号可以有一个可选的轮廓，这是由一个线符号定义的。
