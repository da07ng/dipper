匯聚點

該工具適用於點圖徵圖層（分析圖層）以及區域圖徵的圖層。 首先確定每個區域內的點。 確定點與區域的空間關係後，會計算出區域內所有點的統計數據並將統計數據指定給該區域。 最基本的統計數據是區域內點的計數，但是也可以獲取其他統計數據。

例如，假設以咖啡店位置作為點圖徵，以縣作為區域圖徵，並且希望按縣匯總咖啡銷售額。 假定咖啡店具有 TOTAL_SALES 屬性，則您可以獲得每個縣內所有 TOTAL_SALES 的總和，或者每個縣內 TOTAL_SALES 的最小值或最大值，或每個縣內所有銷售額的標準差。

該結果為一幅地圖，每個區域的顏色是根據點的總數除以區域的面積所得到的結果。 將計數除以面積稱為正規化，並且是製圖人員按面積顯示數據的常用方法。 正規化有助於更理解空間分佈型態，因為對區域進行目測比較時會考慮到區域的大小。

如果勾選 使用當前地圖範圍，則僅分析當前地圖範圍中可見的分析圖層，以及區域圖層中的圖徵。 如果未勾選，則會對分析圖層和區域圖層內的所有圖徵進行分析，包括超出當前地圖範圍的圖徵。

選擇區域

從 web 地圖中選擇一個點要匯聚的區域圖層。 如果 web 地圖中沒有邊界圖層，可以在 ArcGIS Online 中可用的邊界圖層列表中進行選擇。 位於該圖層的區域邊界內的所有點都將計算在內，還可以使用所選的統計方法進行匯總。

保留未含點圖徵的區域

確定不含點的區域是否顯示在結果圖層中。 如果未勾選，則不含點的區域將不會出現在結果圖層中。 如果勾選，則不含有點的區域將會出現在結果圖層中。

新增統計數據

可以根據輸入點圖層的屬性計算各種統計數據，例如總和、平均值、最小值、最大值以及標準差。 結果圖層將為每個已計算的統計數據納入一個新屬性。 透過選擇一個屬性和統計數據（將顯示一個新的下拉式方塊）可新增任意數量的統計數據。

選擇做為分組依據的欄位

可以使用分析圖層中的屬性來建立統計群組。 例如，如果要將犯罪事件匯聚至鄰近地區邊界，可能會有一個含有五種不同犯罪類型的屬性 Crime_type。 各種唯一的犯罪類型構成一個群組，並將針對 Crime_type 的每個唯一值計算您選擇的統計數據。 每個群組的統計數據以及點計數將顯示在結果圖層的快顯中。 選擇分組屬性後，將建立兩個分析結果：結果圖層，以及包含每個唯一群組的統計數據的結果表。

新增少數、多數

選擇作為分組依據的欄位時，將啟用此勾選方塊。 如果想要瞭解每條邊界內每個組中哪些屬性值為少數（所占比例最小）、哪些屬性值為多數（所占比例最大），請選擇 新增少數、多數。 選擇 新增少數、多數後，兩個新欄位將新增到結果圖層。 如果同時選擇 新增百分比，則會有兩個額外欄位新增到結果圖層，其中包含每組內少數屬性值和多數屬性值所占的百分比。

例如，如果您是一位城市規劃者並且想要瞭解所有城市公園中哪些樹種占少數、哪些樹種占多數，則將樹的數據與公園區域匯聚在一起並勾選此選項。

新增百分比

選擇做為分組依據的欄位時，將啟用此勾選方塊。 如果想要尋找每個屬性值在每個群組內所占的百分比，請選擇 新增百分比。 在結果表格新增一個新欄位，其中包含各群組內每個屬性值所占的百分比。 如果同時勾選 新增少數、多數，則會有兩個額外的欄位新增到結果圖層，其中包含每組內少數屬性值和多數屬性值所占的百分比。

結果圖層名稱

在 我的內容中建立並新增到地圖中的圖層的名稱。 預設名稱是根據分析類型以及分析圖層名稱。 如果該名稱的圖層已存在，則系統將要求您提供其他名稱。

使用 將結果儲存到下拉式方塊，可以在 我的內容中指定文件夾的名稱，結果將儲存到該文件夾中。
