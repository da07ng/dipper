概述-JavaScript API

简介

ArcGIS API for JavaScript是由美国Esri公司推出，随ArcGIS 9.3版本发布，是Esri基于dojo框架和REST风格实现的一套编程接口。通过ArcGIS API for JavaScript可以对ArcGIS for Server进行访问，并且将ArcGIS for Server提供的地图资源和其它资源嵌入到WEB应用中。

功能介绍

基础地图服务
基于Map对象实现地图加载。Map对象是ArcGIS API for JavaScript的核心对象。

空间查询服务
利用QueryTask功能类，可以在某个地图服务的子图层内实现空间的属性查询。

空间统计服务
主要用来对查询的结果进行统计，目前支持求和，最小值，平均值，标准差，查询记录个数5种统计。

定位服务
用于提供从地址到经纬度坐标或者从经纬度坐标到地址的转换服务。

专题展示服务-密度图
使用随机分布的点的密度来表现要素某个属性值得大小。也可以对图层的多个属性值着色，通过指定不同的点符号来区分。

专题展示服务-热点图
通过HeatmapRenderer更新FeatureLayer的热点渲染。渲染过程使用标准着色方案，也可以自定义着色方案

标注服务
通过每类符号对应的特定类型（点、线，面和文本）实现地图标注。

分级渲染服务-点
通过点的渲染方式进行分级结果展示。

分级渲染服务-面
通过面的渲染方式进行分级结果展示。

统计信息展示服务
利用统计功能实现信息查询，通过统计信息展示服务在图表中呈现统计值。

时态展示服务
时间滑块控件用于在Web端对时态感知图层提供支持。

格网统计展示服务-网格
网格图是基础的格网图展现形式，利用规则格网样式进行特征展示。

格网统计展示服务-蜂窝
蜂窝图新型格网图展现形式，利用六边形组成的类似蜂窝的网格进行底图展现。

新型聚合图服务
JavaScript与HTML5的技术融合，动态实现要素聚合。

大数据渲染服务
利用丰富的渲染接口，得到大数据分级渲染专题图。
